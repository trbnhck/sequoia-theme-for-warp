## Installation Instructions

### Automatic Setup

1. Run `curl {URL_TO_SCRIPT} | bash`
2. Open Warp
3. Navigate to `Settings` > `Appearance`
4. Click on `Current Theme` and then search for `Sequoia` and either choose between `Moonlight` or `Monochrome`
5. Enjoy!
